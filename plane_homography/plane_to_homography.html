def compute_normal_from_vertices(vertices):
    line1 = vertices[2, :] - vertices[0, :]
    line2 = vertices[3, :] - vertices[1, :]

    normal = np.cross(line1, line2)

    return normal / np.linalg.norm(normal)

def homogenize(points):
    return np.hstack((points, np.ones((points.shape[0], 1))))

def homogenize_pose(P):
    return np.vstack([np.array(P), np.array([[0.0, 0.0, 0.0, 1.0]])])

Homography matrix to transfer pixels from view 1 to view 2 via a plane with params (n, d)¶

$$ {}^2H_1 = K ({}^2R_1 - \frac{{}^2t_1.n^T} {d}) K^{-1} $$

Note to use the formula:

  1. Plane parameters should be in view 1
  2. The direction of n and d should be w.r.t camera 1 and are oriented as per below figure:
